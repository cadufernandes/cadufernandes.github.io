Configurando Github no windows
Programas utilizados: -Sublime(editor de texto): http://www.sublimetext.com/ -Gitbash(conexão com github) http://git-scm.com/downloads
Site visitados: -https://github.com/ Instalação do Gitbash: Instalação é fácil, pois não utilizaremos nenhum configuração diferenciada, iremos utilizar a instalação padrão. Execute o programa em seguida na tela de componentes deixei a configuração default, depois NEXT Na tela de configuração "Path environment" também mantenha a opção default "Use Git Bash only", caso deseja utilizar outras opções, leia-as com atenção e escolha como quiser, depois NEXT Na próxima tela, escolha a primeira opção: Checkout Windows..., NEXT novamente. depois é só concluir. Configuração do Gitbash: Abra o GitBash e digite: Entre com esses codigos para configurar o Usuario e o emails que será configurado. $ git config --global user.name "Nome Completo" $ git config --global user.email "email@domain.com" Gerar SSH: Criar as chaves publica e privada: Comando: ssh-keygen -t rsa -C "your_email@domain.com" Depois acesse o site do github, efetue o login, depois clique em opções/Settings, clique em SSH Keys e em seguida Add SSH Key. No arquivo gerado pelo prompt, pegue a chave gerada no conteudo do mesmo e copie no campo KEY do site depois clique em Add Key. Para verificar se funcionou digite no prompt: ssh -T git@github.com (usar o email q vc cadastrou) Comandos importantes: Git Status: veirifa a situação dos codigos, quais foram upados no site e quais não. Git add : adiciona o arquivo para upar. Git commit -m "": comita o arquivo no site e cria uma mensagem de identificação
